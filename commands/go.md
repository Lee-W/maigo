# /maigo:go

> 「It's MyGO!!!!!」

把這件事交給 MyGO!!!!!。從前奏到尾聲，五個人各自負責自己那一段。

## 使用

```
/maigo:go <任務描述>
```

## 流程

完整流程依 [`skills/teammate-flow`](https://github.com/Lee-W/maigo/blob/main/skills/teammate-flow/SKILL.md)——
🐱 樂奈探索 → 🩵 燈寫 plan → 使用者確認 → 🎀 愛音實作 → 🟡 爽世 review → 🟣 立希驗證。

審查與驗證為**順序**進行（先 🟡 爽世擋，通過後才跑 🟣 立希）。
